# Kursova Project — Інтернет-магазин 
Виконав:Пташников Василь AI-235

- Цей проєкт є курсовою роботою, що представляє собою сучасний веб-додаток для електронної комерції, побудований на стеку ASP.NET Core MVC.

## Технологічний стек
- **Framework**: ASP.NET Core 9.0 (MVC)
- **Database**: SQLite + Entity Framework Core (для забезпечення портативності та легкого запуску без налаштування серверів БД).
- **Authentication**: ASP.NET Core Identity (RBAC — Role-Based Access Control).
- **Payments**: Stripe API + Webhooks.
- **Frontend**: Bootstrap 5, Razor Pages.

## Додаток А: Структура проєкту
Нижче наведено опис ключових директорій та файлів проєкту:

- **Kursova/Controllers/** — містить класи-контролери, які відповідають за обобку запитів (товари, кошик, оплата).
- **Kursova/Models/** — описує сутності бази даних (Product.cs, ViewModels) та структуру даних.
- **Kursova/Views/** — Razor-шаблони (HTML сторінки) для відображення інтерфейсу користувача та адмін-панелі.
- **Kursova/Data/** — містить контекст бази даних (DbContext) та файл Seeder для початкового заповнення товарів.
- **Kursova/Migrations/** — історія змін структури бази даних.
- **Kursova/wwwroot/** — статичні файли проєкту: CSS-стилі, JavaScript-скрипти та зображення.
- **Kursova/kursova.db** — файл локальної бази даних SQLite.
- **README.md** — файл з описом проєкту та інструкцією для запуску.

## Інструкція для запуску
1. Переконайтеся, що у вас встановлено **.NET 9 SDK**.
2. Відкрийте термінал у кореневій папці проєкту.
3. Виконайте команду для відновлення залежностей:
   ```bash
   dotnet restore
   ```
4. Запустіть проєкт:
   ```bash
   dotnet run --project Kursova
   ```
5. Проєкт буде доступний за адресою `https://localhost:5001` (або іншою, вказаною в консолі).

## Тестування
Проєкт містить повний цикл тестування функціоналу:
- Реєстрація та авторизація користувачів.
- Керування товарами (Admin CRUD).
- Додавання товарів у кошик.
- Процес оформлення замовлення через Stripe.
- Перевірка прав доступу (RBAC) з коректною обробкою сторінки "Access Denied".
