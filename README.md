# Інтернет-магазин продуктів харчування

Курсова робота на тему: "Проектування та розробка інформаційної системи інтернет-магазину".

## Опис проєкту
Веб-застосунок для онлайн-торгівлі продуктами харчування з інтегрованою платіжною системою Stripe. Система побудована на архітектурі ASP.NET Core MVC і забезпечує повний цикл покупки: від перегляду каталогу до оплати карткою.

## Основні можливості
- **Каталог:** Відображення товарів за категоріями, пошук та фільтрація.
- **Кошик:** Динамічне управління товарами (додавання, зміна кількості, видалення).
- **Авторизація:** Рольова модель (Адміністратор та Клієнт).
- **Адмін-панель:** Керування асортиментом товарів (CRUD).
- **Оплата:** Безпечні онлайн-платежі через Stripe API (валюта UAH).

## Стек технологій
- **Back-end:** .NET 8, ASP.NET Core MVC.
- **ORM:** Entity Framework Core (SQLite).
- **Security:** Microsoft Identity.
- **Payments:** Stripe.net.
- **Front-end:** HTML5, CSS3 (Bootstrap 5), JavaScript.
